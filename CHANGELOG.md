# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-03-06

### Added
- Community follow scope and social feed improvements, including comments and following-only feeds.
- Saved prompts collections with create, rename, delete, and collection membership endpoints.
- Dashboard history and API key filtering/export utilities with focused unit coverage.
- Recommendation ranking improvements and expanded related prompt handling.
- Route-guard verification and broader smoke coverage for phase 4 social flows.

### Changed
- Discovery and category filtering behavior to improve relevance and consistency.
- Public page SEO and accessibility improvements.
- Build/test pipeline hardening with verify and smoke flows.

### Fixed
- Phase 4 smoke preflight issues and flaky checks.
- Prompt dedupe protections and migration hardening for community and auth flows.
